# etl-process

## How to run

### Clone this source repo

```bash
$ git clone https://github.com/mateusrfon/etl-process.git
```

### Go to repo folder

### Install dependencies

```bash
$ npm i
```

### Build

```bash
$ npm run build
```

### Start

```bash
$ npm run start
```

### Developer start

```bash
$ npm run dev
```

### Run tests

```bash
$ npm run test
```
